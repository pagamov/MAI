{\bfseries Вариант сортировки:} Сортировка подсчётом.

{\bfseries Вариант ключа:} { \normalfont\ttfamily Числа от 0 до 65535 включительно. }

{\bfseries Вариант значения:} { \normalfont\ttfamily Строки длинны до 2048 знаков включительно.}
}
