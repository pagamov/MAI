\CWHeader{Курсовой проект}

\CWProblem{

Разработать алгоритм А* для поиска минимального пути в графе.

Формат входных данных:

(int) start - номер стартовой вершины.
(int) end - номер конечной вершины.

(int) n - число вершин в графе. 
(int) m - число ребер в графе.

Далее в цикле по кол-ву вершин вводятся их координаты x и y в интах.

Далее в цикле по кол-ву ребер вводятся from, to и waight, для задания стоимости ребер графа.

Формат вывода:

Путь от начала до конца в виде номеров посещенных вершин, если такой путь есть.

Если пути нет, выводится -1.

}
